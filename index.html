Git ve GitHub Hakkında 10 Madde

Git ile İlgili Maddeler

  • 1. Git nedir? Versiyon kontrol sistemidir.
  • 2. Yerel çalışır: İnternet olmadan da kullanılabilir.
  • 3. Commit yapısı: Değişiklikler commit ile kaydedilir.
  • 4. Branch kullanımı: Farklı özellikler için dallar açılır.
  • 5. Merge ve Conflict: Branch'ler birleştirilirken çakışmalar olabilir.

GitHub ile İlgili Maddeler

  • 6. GitHub nedir? Git projelerini barındıran platformdur.
  • 7. Remote Repository: Projenin uzaktaki kopyasıdır.
  • 8. Pull Request: Değişiklik teklifleri için kullanılır.
  • 9. Issue sistemi: Hata ve görev takibi yapılır.
  • 10. Açık kaynak katkısı: Başka projelere katkı sağlanabilir.
